In this first graph our school teacher friend is kindly pointing out the lack of correlation between these variables. Throughout these graphs, the bubble size reflects the number of cars listed for sale.

It is worth noting that in the NT vehicle prices are the second highest despite it having the lowest average taxable incomes.

Overall, this graph does not help us establish a link between vehicle prices and incomes. 


Let's look at some tax returns


When we introduce the average amount claimed by each person for car related expenses, we suddenly see a correlation with car prices. Makes sense.

 When running some regression analysis the R2 value comes back at .3994 indicating a weak correlation.

There are too many other variables which would need to be factored in to form a conclusive opinion about this relationship; but this result suggest further investigation may be justified.
