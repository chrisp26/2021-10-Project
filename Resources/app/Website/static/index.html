Overview

The goal of this project is to investigate and compare any correlations between used vehicles listings and financial data from the ATO. I want to see if the income within an Australian suburb leads to higher prices on used vehicles.

Easy.


Data Sources
  1. Vehicle listings scrapped from carsguide.com.au
  2. 2016 Tax data downloaded from the ATO website 




How I thought I was going to do things...


  1. Scrape carsguide website and store data in a Mongo database
  2. Cross reference the above with ABN data and create collections of objects containing data on each suburb.
  3. Use D3 and Leaflet to throw it all into an interactive graph allowing the user to plot different data sets and view any relationships.




What actually happened...


  1. Scraped carsguide website....
